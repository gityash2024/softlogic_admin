# softlogic_admin

SoftLogic Admin Console built with React, Vite, TypeScript, Tailwind CSS, Radix UI, Recharts, and the SoftLogic backend APIs.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production frontend uses:

```bash
VITE_API_BASE_URL=https://softlogic-whiteboard-backend-testin.vercel.app/api/v1
```

If the environment variable is not set, the app falls back to the same testing backend URL.
